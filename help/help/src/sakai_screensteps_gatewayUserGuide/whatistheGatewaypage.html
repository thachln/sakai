What is the Gateway page?

The Gateway page is the login page for your Sakai instance.

You will see the following information displayed on the Gateway page:

  1. Login fields - Enter your username and password to log in to the system.
  2. Gateway tools - A list of tools available to users who are not yet authenticated in the system.
  3. Message of the Day - System-wide messages posted by Sakai administrators appear here.
  4. Welcome - A system-wide message displays to welcome new users.
