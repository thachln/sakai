How do I share a rubric?

Go to the Rubrics tool.

Select the Rubrics tool from the Tool Menu of your site.

Select the Make Public icon for the rubric you would like to share.

The icon will change to a globe to indicate that it is public.

The rubric will now appear listed in the Publicly Shared Rubrics area.

The rubric is now shared throughout your institution. Any user with the appropriate site permissions may copy the shared rubric into his or her site to use for grading or to make changes to their individual copy.
